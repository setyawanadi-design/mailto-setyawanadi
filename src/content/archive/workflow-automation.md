## The idea

I've been thinking about how to automate some of the repetitive reporting tasks in field operations. Right now, a lot of the work involves:

- Manually compiling daily activity reports
- Cross-referencing inventory counts across sites
- Tracking team deployment schedules in spreadsheets

## What I'm building

A simple dashboard that pulls data from multiple sources and generates daily reports automatically. The stack I'm considering:

- **Google Sheets API** for data input (teams are already comfortable with Sheets)
- **Node.js** backend for data processing
- **Simple web dashboard** for visualization
