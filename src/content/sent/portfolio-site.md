Hi there,

I wanted to share my latest project: **setyawanadi.design** — the very site you're browsing right now!

## Project Overview

A personal portfolio and blog built to look and feel like Mozilla Thunderbird, the email client. The concept is simple: your personal content (blog posts, projects, about page) is presented as "emails" in a familiar three-panel layout.

## Tech Stack

| Technology | Purpose |
|-----------|---------|
| **Astro** | Static site generation |
| **Vanilla CSS** | Custom design system |
| **Content Collections** | Typed content management |
| **SVG Icons** | Lightweight, crisp icons |
| **Inter Font** | Modern typography |

## Design Features

- **Three-panel layout** — Sidebar, message list, reading pane
- **Dark theme sidebar** — Purple accents inspired by Thunderbird
- **Light reading pane** — Optimized for content readability
- **Responsive design** — Adapts to mobile and tablet
- **Email metaphor** — Folders map to content sections

## What I Learned

Building this project taught me a lot about CSS Grid layouts, Astro's content collections system, and the art of making a novelty concept feel genuinely usable.
